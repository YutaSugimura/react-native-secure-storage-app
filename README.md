# Securely manage private keys using React-native-keychain


## Get Started

1. Pod install

```zsh
  cd ios
  USE_FABRIC=1 RCT_NEW_ARCH_ENABLED=1 pod install

```

2. Node Start

```zsh

  yarn start
```

3. Start Emulators

```zsh

  yarn ios
  # or
  yarn android
```
